//**********    EMAIL GAME   ***********//


***START***

- creiamo una lista di email.
- chiediamo all'utente una email.
- **SE NON** è inserita nella lista:
    - diciamo all'utente che non può entrare.
- **SE** è inserita nella lista:
    - diciamo all'utente che è il benvenuto.

***END**
